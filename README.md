# Jupyter Notebook (jupyter-notebook)
The Jupyter Notebook is an open-source web application that allows you to create  and share documents that contain live code, equations, visualizations and narrative text.

## APIs

### Jupyter Notebook REST API
REST API for managing and interacting with Jupyter Notebook servers, including kernels, sessions, contents, and terminals.


**Human URL:** [https://jupyter-notebook.readthedocs.io/](https://jupyter-notebook.readthedocs.io/)


#### Tags:

 - kernels, sessions, notebooks, files

#### Properties

- [X-documentation](https://jupyter-notebook.readthedocs.io/en/stable/rest_api.html)
- [X-openapi](https://petstore.swagger.io/?url=https://raw.githubusercontent.com/jupyter/notebook/master/notebook/services/api/api.yaml)
- [X-github](https://github.com/jupyter/notebook)

### Jupyter Kernel Gateway API
Web server that provides headless access to Jupyter kernels for remote execution of code.


**Human URL:** [https://jupyter-kernel-gateway.readthedocs.io/](https://jupyter-kernel-gateway.readthedocs.io/)


#### Tags:

 - kernel, gateway, execution

#### Properties

- [X-documentation](https://jupyter-kernel-gateway.readthedocs.io/en/latest/)
- [X-github](https://github.com/jupyter/kernel_gateway)

### JupyterHub REST API
Multi-user server management API for spawning, managing, and proxying multiple instances of single-user Jupyter notebook servers.


**Human URL:** [https://jupyterhub.readthedocs.io/](https://jupyterhub.readthedocs.io/)


#### Tags:

 - hub, multi-user, authentication

#### Properties

- [X-documentation](https://jupyterhub.readthedocs.io/en/stable/reference/rest-api.html)
- [X-openapi](https://raw.githubusercontent.com/jupyterhub/jupyterhub/master/docs/rest-api.yml)
- [X-github](https://github.com/jupyterhub/jupyterhub)
